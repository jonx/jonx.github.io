NonoGotchi

BETA TEST

Thanks for testing NonoGotchi! This is an early beta for Nothing Phone users.

What is it?

A Tamagotchi-style virtual pet that lives on your phone's Glyph display (the LED matrix on the back).

How to use

  • Feed, clean, play, and pet your creature using the buttons
  • Shake your phone to wake up your pet
  • The pet grows through stages: Egg → Baby → Child → Teen → Adult
  • Watch the weather sync with your location

Download

Download Release APK Recommended for most testers (~40 MB) Download Debug APK Debug screens & features enabled (~56 MB)
Installation:
  1. Contact me to get a beta access code
  2. Download the APK file
  3. Open the file on your Nothing Phone
  4. If prompted, enable "Install from unknown sources"
  5. Complete the installation
  6. Enter your access code when prompted

What to test

  • Does the Glyph display work properly?
  • Is the pet responsive and animations smooth?
  • Try the new "Weather on Glyph" toggle in Settings
  • Let your pet get hungry/tired and see how it reacts
There's a secret easter egg hidden in the app...
Known issues:
  • This is pre-release, expect bugs
  • Database resets may happen between updates
Privacy:

No data is collected or shared, at all. The app only uses internet access to retrieve weather information based on the location you manually enter.

Feedback
